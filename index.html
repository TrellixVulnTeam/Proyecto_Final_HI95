Portfolio - Diseño de Paginas Web

Mi primer proyecto hecho solamente en HTML y con nociones basicas de CSS, en este proyecto se aprendio lo basico para poder seguir avanzando.

El segundo proyecto que realice estudiando en CODERHOUSE en el cual ya estaba aplicando FLEXBOX y GRID para acomodar los objetos y darle dimension a la pagina.

Este proyecto fue el tercero que realice y fue donde apique por primera vez el conocimiento sobre Bootstrap.
